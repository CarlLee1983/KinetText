## Phase 4 MP4 轉換

```
輸入: 合併的 MP3 檔案
  ↓
MP4ConversionService.convert()
  ├─ 參數: 輸入路徑、輸出路徑、元資料
  ├─ FFmpeg: MP3 → M4A (AAC 編碼)
  └─ 驗證: music-metadata 讀取元資料
  ↓
輸出: M4A 檔案
```

## 完整管道流程

```
Phase 1: RetryService (失敗重試)
  ↓
Phase 2: AudioConvertService (WAV → MP3)
  ↓
Phase 3: AudioMergeService (合併 MP3)
  ↓
Phase 4: MP4ConversionService (MP3 → M4A)
  ↓
最終產物: M4A 音頻檔
```

  ```typescript
  test("完整管道 E2E: WAV → MP3 → 合併 → M4A", async () => {
    // 1. 準備輸入
    const inputFiles = await fixtures.getSampleWAVs(3)

    // 2. Phase 2: 轉換
    const mp3Files = []
    for (const file of inputFiles) {
      const mp3 = path.join(tmpDir, "phase2", `converted.mp3`)
      const result = await audioConvertService.convert(file, mp3)
      expect(result.success).toBe(true)
      mp3Files.push(mp3)
    }

    // 3. Phase 3: 合併
    const mergeResult = await audioMergeService.mergeBatch({
      files: mp3Files,
      targetDuration: 39600,
      outputDir: path.join(tmpDir, "phase3"),
    })
    expect(mergeResult.groups.length).toBeGreaterThan(0)

    // 4. Phase 4: 轉換
    const m4aFiles = []
    for (const mergedMp3 of mergeResult.groups[0].mergeResult.outputPaths) {
      const m4a = path.join(tmpDir, "phase4", "output.m4a")
      const result = await mp4ConversionService.convert(mergedMp3, m4a, {
        title: "完整管道測試"
      })
      expect(result.success).toBe(true)
      m4aFiles.push(m4a)
    }

    // 5. 驗證
    expect(m4aFiles.length).toBeGreaterThan(0)
    for (const m4a of m4aFiles) {
      expect(await Bun.file(m4a).exists()).toBe(true)
      const meta = await parseFile(m4a)
      expect(meta.format?.container).toBe("M4A")
    }
  })
  ```
