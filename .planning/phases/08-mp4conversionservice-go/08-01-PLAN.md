**converter.go** (80-120 lines):
- ConvertMP4(ctx context.Context, req *MP4ConvertRequest) (*MP4ConvertResponse, error) function
  - Create ffmpeg input: `ffmpeg.Input(req.InputFile, ffmpeg.KwArgs{"-loglevel": "quiet"})`
  - Build M4A KwArgs via buildM4AKwArgs(req.Bitrate) — set c:a=aac, b:a={bitrate}k, movflags=+faststart
  - Build metadata args via buildMetadataArgs(req.Metadata) — return []string of -metadata key=value pairs
  - Chain output: `.OutputArgs(append(buildMetadataArgs(...), "-y")).Output(req.OutputFile).OverWriteOutput().Run(ctx)`
  - Catch error: if err != nil return &MP4ConvertResponse{Success: false, Error: err.Error()}
  - Return success response with OutputFile

- buildM4AKwArgs(bitrate int) ffmpeg.KwArgs function:
  - If bitrate <= 0, set default 256 (AAC standard)
  - Return KwArgs{"c:a": "aac", "b:a": fmt.Sprintf("%dk", bitrate), "movflags": "+faststart"}

- buildMetadataArgs(metadata *MP4MetadataGo) []string function:
  - If metadata nil, return empty []string
  - Build -metadata key=value pairs for each non-empty field:
    - title → "title"
    - artist → "artist"
    - album → "album"
    - date → "date"
    - genre → "genre"
    - trackNumber → "track" (special: format as integer)
    - comment → "comment"
  - Note: No escaping needed (ffmpeg-go passes array args, not shell strings)
  - Return []string of all pairs

**main.go** (20-30 lines):
- func main() entry point
- Read stdin as JSON: json.NewDecoder(os.Stdin).Decode(&req) into MP4ConvertRequest
- Validate req: check InputFile exists, OutputFile path writable
- Call ConvertMP4(context.Background(), &req)
- Marshal response as JSON: json.NewEncoder(os.Stdout).Encode(response)
- Errors written to response.Error, not stderr (keep stdout clean)
