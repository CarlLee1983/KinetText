## 技術決策概要

| 決策項 | 選擇 | 理由 |
|--------|------|------|
| **IPC 協議** | JSON subprocess | 跨平台穩定、易除錯、易隔離 |
| **FFmpeg 綁定** | ffmpeg-go | API 簡潔、文檔完善、Go 原生 |
| **進程管理** | 無狀態（按需） | 簡化複雜度，啟動開銷 ~1ms 可接受 |
| **後端切換** | 配置化標誌 useGoBackend | 向後相容，漸進遷移 |
| **重試策略** | RetryService + 錯誤分類 | 複用 Milestone 1 成果 |
